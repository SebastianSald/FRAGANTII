Prompt Optimizado para Replit y Vercel
Al configurar el proyecto, crea siempre un archivo vercel.json en la raíz para asegurar el despliegue correcto de una SPA (Single Page Application) con Vite.
El archivo debe tener exactamente esta configuración:

json
{
  "framework": "vite",
  "outputDirectory": "dist/public",
  "rewrites": [
    { "source": "/(.*)", "destination": "/index.html" }
  ]
}
No esperes a que falle el deploy, inclúyelo desde el principio.
Asegúrate también de que el vite.config.js tenga base: "./" y que el build apunte a "dist/public" para compatibilidad total con Vercel.

🌹 Construye un sitio web elegante y sensorial para "FRAGANTI"

Elementos gráficos: curvas fluidas, transparencias, reflejos sutiles y degradados acorde a la paleta de colores

Animaciones: transiciones suaves tipo fade-in y parallax en scroll; micro-interacciones al hover sobre productos
